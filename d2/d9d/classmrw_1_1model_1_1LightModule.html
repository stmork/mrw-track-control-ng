Detailed Description

This module connects light sources such as LEDs or light bulbs.

The lighting up may follow different profiles using PWM of the controlling CAN controller. The lighting starts when the sensor values fall below the threshold. The light turns off when the sensor values go beyound the threshold + 4 (hysteresis).

See also
https://github.com/stmork/mrw/wiki/LichtProfile
