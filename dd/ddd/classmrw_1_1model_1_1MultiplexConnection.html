Detailed Description

This class represents a serial multiplex connection module.

Since this is not connected to the 16-pin IO port of the CAN Controller its connected using the 10-pin serial connector. It is a cost and space saving method to address up to 64 light sources such as LightSignals. Espacially an outgoing main light signal takes five pins and an additional pre light signal may used four pins more.
