Detailed Description

This class represents a switch module which is connected to a CAN controller using one or two 16-pin connectors depending on an existing end response is available inside the inductive actuator.

Note
Even the module is called switch module for historical reasons it can drive also form signals with inductive actuators.

A section module can control up to four switches. It can be mixed with form signals which may need three control lines for the inductive actuators.
