◆ isRightBended()

virtual bool mrw::ctrl::BaseSwitchController::isRightBended ( ) const
pure virtualinherited

This method returns true if the switch should be drawn right handed.

Note
It is possible to draw a right bended switch which is in reality left handed. So you should distinguish between left/right handed against left/right bended.
There is no isLeftBended() method which is the opposite of this method.
Returns
True if the switch should be drawn right bended.
