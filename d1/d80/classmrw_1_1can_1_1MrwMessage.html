Detailed Description

This class represents a CAN bus frame in model railway manner.

It may have two message types:

  1. As a command to be sent.
  2. As a response from a previous command.

A command may address all mrw::model::Controller instances using the broadcast ID CAN_BROADCAST_ID. In case a mrw::model::Device is addressed both is needed: the mrw::model::Controller ID and the mrw::model::Device unit number.

See also
mrw::model::Controller:id()
mrw::model::Device::unitNo()
