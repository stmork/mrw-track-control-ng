Detailed Description

This class is a unique way to create a std::string from any binary data representation.

Only implement the pure virtual method toString().

QString String::format ( const QString & code,
const QString & input )
staticnoexcept

This method surrounds a string with a given console format command.

After the given input string to format command is reset.

Note
The final ALL_OFF formatting code deactivates all previous codes so you have to prepare the strings as your needs by multiple calls to this method.
const QString & result =
String::format(RED_ON + ITALIC_ON, "red italic") +
String::format(ITALIC_ON, " only italic");
