Detailed Description

template<class T>
class mrw::util::Singleton< T >

This template class implements the singleton pattern as a base class for real implementations.

Attention
Do not derive this class more than once since the instance() method will only work with the first derived class.

The default constructor should only be accessible to derviced classes.

Derived singleton classes should look like the following example:

class ExampleSingleton : public Singleton<ExampleSingleton>
{
// private default constructor!!!
ExampleSingleton() = default;
friend class Singleton<ExampleSingleton>;
public:
// ... any accessible members.
};
