Detailed Description

template<class T>
class mrw::util::ConstantEnumerator< T >

This convenience class provides a mapping from a symbolic value to its string representation.

It is used for printing clear text value while debugging.

The following example shows how to map error codes to their QString representation:

flag_map TestUtil::map
{
CONSTANT(EINVAL),
CONSTANT(EAGAIN),
CONSTANT(EBUSY)
};

You may access the mapper like every std::unordered_map.

Note
The operator[] may throw a std::out_of_range exception. It is a safe way to map using the get() method instead.

◆ findKey()

template<class T >
ConstantMapConstIterator mrw::util::ConstantEnumerator< T >::findKey ( const QString & value) const
inlinenoexcept

This method is a reverse lookup to find a key from its value.

The return value is an const_iterator to a ConstantMapPair type. If this iterator is equal to end() no key was found. Otherwise you can select the key by using

auto it = mapper.findKey("WHAT_EVER");
if (it != mapper.end())
{
auto key = it->first;
}
Parameters
valueThe value from which the key should be reverse lookedup.
Returns
A ConstantMapPair const_iterator which may contain the key when the const_iterator is not equal to the end() iterator.
